BEM-VINDO!

Este site foi desenvolvido para colaborar com o evento da Feira Das Profissões da EEEP Dr. Napoleão Neves Da Luz. Ele tem o objetivo de apresentar o Curso de Informática aos novos estudantes que ingressarão na escola em 2024.

Desde já, o Primeiro Ano de Informática dá as Boas Vindas a todos que entrarão para a família Dr. Napoleão e os convida a entrar para o Curso de Informática.

SOBRE O CURSO

O curso de informática pode ser bastante desafiador no começo, onde é necessário aprender a Lógica de Programação, ou seja, como o computador interpreta os comandos dados a ele. Além de ter que aplicar esse conhecimento em seguida para dominar a primeira Linguagem de Programação. Porém, esses conhecimentos serão muito úteis na vida, tanto para ingressar no mundo de trabalho, onde o conhecimento de informática está sendo cada vez mais valorizado, tanto para a sua vida cotidiana, onde você poderá entender melhor os avanços da tecnologia e como eles podem afetar sua vida.

Para as aulas práticas, a escola conta com um laboratório com bons computadores. Segue a imagem do próprio a seguir.

O QUE VAMOS APRENDER?

HTML - CSS - JAVASCRIPT

Aprenderemos a como criar sites usando as linguagens HTML, CSS e JAVASCRIPT. Este site é um exemplo de um projeto básico feito utilizando essas linguagens.

PYTHON

Aprenderemos a programar em PYTHON. Que pode ser usado para automatizar tarefas e preencher formulários, além de ser uma das principais linguagens usadas por IAs.

ARQ. E MANUT. DE PC

Aprenderemos sobre como funcionam os componentes dos computadores. Entenderemos como funcionam as memórias, fontes, processadores e etc. Além de também estudarmos robótica.
